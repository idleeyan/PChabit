## 开发环境安装计划

### 需要安装的组件

#### 1. .NET SDK 9.0（必需）
- **下载地址**: https://dotnet.microsoft.com/download/dotnet/9.0
- **安装命令**: `winget install Microsoft.DotNet.SDK.9`
- 项目 global.json 要求 SDK 9.0.0

#### 2. Visual Studio 2022 Community（推荐）
- **下载地址**: https://visualstudio.microsoft.com/downloads/
- **安装命令**: `winget install Microsoft.VisualStudio.2022.Community`
- **必需工作负载**:
  - .NET 桌面开发
  - Windows 应用程序开发 (Windows App SDK)

#### 或替代方案: VS Code + .NET SDK
如果不想安装完整的 Visual Studio，可以使用：
- VS Code: `winget install Microsoft.VisualStudioCode`
- C# Dev Kit 扩展
- 但 WinUI 3 开发体验不如 Visual Studio 完整

### 安装步骤

1. **安装 .NET SDK 9.0**
   ```powershell
   winget install Microsoft.DotNet.SDK.9
   ```

2. **安装 Visual Studio 2022 Community**
   ```powershell
   winget install Microsoft.VisualStudio.2022.Community
   ```

3. **在 Visual Studio Installer 中选择工作负载**:
   - ".NET 桌面开发"
   - "Windows 应用程序开发"

4. **验证安装**
   ```powershell
   dotnet --version
   dotnet build src/Tai.App/Tai.App.csproj
   ```

### 预计安装时间
- .NET SDK: ~5 分钟
- Visual Studio: ~20-30 分钟
