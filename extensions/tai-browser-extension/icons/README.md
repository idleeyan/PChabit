# Tai Browser Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Icon Design Guidelines

The icon should represent the Tai-AI brand:
- Primary color: #667eea (gradient to #764ba2)
- Style: Modern, minimal, rounded corners
- Symbol: Letter "T" or abstract activity tracking symbol

## Creating Icons

You can create icons using:
1. Figma / Adobe Illustrator
2. Online tools like Canva
3. AI image generators
